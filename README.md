# Receipt Tracker — FuturePark

ระบบติดตามใบเสร็จและยอด Roboflow สำหรับร้านค้าในโครงการ FuturePark  
เชื่อมต่อ Google Sheets เป็น database ผ่าน Google Apps Script

## 🌐 เว็บไซต์

**[pattaraporn-rgb.github.io/Receipt-Tracker-FuturePark/receipt-tracker-v2.html](https://pattaraporn-rgb.github.io/Receipt-Tracker-FuturePark/receipt-tracker-v2.html)**

## ✨ ฟีเจอร์หลัก

- Dashboard แสดงสถิติร้านค้า ยอดใบเสร็จ และ Roboflow progress
- อัพโหลดข้อมูลผ่าน Excel template (3 แบบ)
- กรองร้านค้าตามเงื่อนไขต่างๆ เช่น Priority สูง, ยังไม่อัพ RF, อัพครบแล้ว
- ตรวจจับร้านค้าชื่อซ้ำ
- ป้องกันการแก้ไขด้วย PIN
